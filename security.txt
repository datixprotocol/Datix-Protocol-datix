# datix Protocol Security Policy

If you discover a security vulnerability in the datix Protocol smart contract or infrastructure, please report it responsibly.

## Reporting Process

1. Email security@datixprotocol.com with details
2. Include: vulnerability description, affected component, reproduction steps
3. We will acknowledge within 24 hours
4. We aim to patch critical issues within 7 days

## Scope

In scope:
- datix Protocol Solana program (5dpw6KEQPn248pnkkaYyWfHwu2nfb3LUMbTucb6LaA8G)
- datix API (api.datixprotocol.com)
- datix website (www.datixprotocol.com)

Out of scope:
- Third-party integrations
- Social engineering attacks
- Denial of service attacks

## Safe Harbor

We will not pursue legal action against security researchers who:
- Act in good faith
- Avoid privacy violations, data destruction, or service disruption
- Give us reasonable time to fix issues before public disclosure

Thank you for helping keep datix Protocol secure.
